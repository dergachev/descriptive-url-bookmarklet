# Descriptive URL bookmarklet The _Descriptive URL_ bookmarklet is helpful if the URL you would like to paste is too short. It makes a descriptive URL for the current page. To [install the bookmarklet](https://raw.github.com/dergachev/descriptive-url-bookmarklet/gh-pages/install-bookmarklet-screencast.gif), drag the following link to your bookmark toolbar:
Descriptive URL
## How to use it Go to a site with overly short URLs (see examples below), and click on the bookmarklet to append the page title to the end of the URL.
